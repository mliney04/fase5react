Responda las siguientes preguntas

1. ¿Qué es el modelo de caja (Box Model) en CSS y cómo afecta a la presentación de los elementos en una página web?

a. Es una representación visual que se utiliza en diseño web para entender cómo se estructuran y presentan los elementos de una página.
b. Se refiere al área dentro de la caja donde se muestra el contenido real del elemento.
c. Espacio entre el contenido de un elemento y su borde.

2. ¿Cuáles son los componentes principales del modelo de caja y cómo se relacionan entre sí?

a. El contenido, el padding, el borde y el margen.
b. El inicio, el desarrollo, solucion y el final.
c. El contenido, el inicio, margen y conclusion.

3. ¿Cuál es la diferencia entre el borde y el contenido en CSS?

a. El border se utiliza en el modelo de caja y el contenido no.
b. El contenido es lo que hay dentro del elemento y el border es el espacio.
c. Que el contenido se refiere a lo que está dentro del elemento, mientras que el borde es la línea que rodea ese contenido.

4. ¿Cuál es el propósito del padding en una caja?

a. Es la linea que rodea un elemento.
b. Es proporcionar un espacio adicional entre el contenido de la caja y sus bordes.
c. Proporciona el color del modelo de caja.

5. ¿Cómo se calcula el tamaño final de un elemento considerando el modelo de caja y sus propiedades relacionadas?

a. El tamaño final de un elemento sse calcula multiplicando el contenido, el relleno, el borde y el margen.
b. El tamaño final de un elemento se calcula sumando el contenido, el relleno, el borde y el margen.
c. El tamaño final de un elemento se calcula dividiendo el contenido, el relleno, el borde y el margen.
